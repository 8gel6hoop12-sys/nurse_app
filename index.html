看護アシスタント（GUI）ランチャー

※ このページは デスクトップGUI へ誘導するためのポータルです（アプリ本体の処理は一切クラウドへ送信しません。Ollamaローカル固定）。

1) いますぐ起動（環境がある方向け）

Windows：起動バッチをダウンロード macOS：起動スクリプトをダウンロード

ダウンロード後に実行すると python nurse_app.py を起動します。初回は必要パッケージのインストール案内が表示される場合があります。

2) はじめての方（セットアップ手順）

Windows

  1. Python 3.10+ をインストール（未導入なら python.org）
  2. このリポジトリをダウンロード（もしくは git clone ...）
  3. フォルダ内で pip install -r requirements.txt
  4. start_win.bat を実行（または python -X utf8 nurse_app.py）

macOS

  1. Python 3.10+ を用意（例：brew install python）
  2. このリポジトリを入手
  3. フォルダ内で pip3 install -r requirements.txt
  4. chmod +x start_mac.command → ダブルクリック（または python3 -X utf8 nurse_app.py）

※ 追加ライブラリは GUI 本体に必要な PyQt5 と、Excel閲覧用の pandas/openpyxl（任意）です。

3) よくあるつまずき

  • 画面が出ない：Pythonのバージョン/仮想環境を確認。pip show PyQt5 で導入確認。
  • Ollama未導入：アプリ内で案内ダイアログが出ます。指示通りに進めると自動で入ります。
  • Excelが見えない：pandas/openpyxl を入れるか、nanda_db.xlsx の配置を確認。

セキュリティ上、ブラウザからデスクトップアプリを自動起動することはできません（カスタムURLスキーム等の導入が別途必要）。このページは「ダウンロード＆起動」を簡単にするためのポータルです。

ステータス

ローカル起動チェックは不要です。上の手順で実行してください。
